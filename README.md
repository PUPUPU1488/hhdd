программа использующая паттерн Фасад

Задание: написать стратегию где есть две сущности - царь и народ. Царь может объявлять войну с соседями, собирать налоги, издавать указы и т.д..
На каждое действие царя народ реагирует по разному, настроение народа может подниматься-повышаться, также от действий царя может меняться уровень налогов и размер казны.
